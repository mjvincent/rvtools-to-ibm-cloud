# Carbon Private Workbook Evidence Template

Use this template for customer-scale Carbon performance evidence. Do not include
customer names, VM names, host names, IP addresses, network names, owners,
application names, workbook filenames, or local file paths.

## Evidence Run

```text
Date:
Carbon branch/commit:
Environment:
Workbook label:
Workbook size category:
Assignment rows:
Summary parse elapsed:
Threshold:
Result:
HTTP status:
Notes:
```

## Safe Labels

Use neutral labels such as:

- medium-wave
- large-estate
- customer-a-medium
- customer-b-large

## Unsafe Content

Do not record:

- workbook file paths or filenames
- company or customer names
- VM, host, cluster, datastore, or network names
- IP addresses or CIDR ranges
- owner or application names
- screenshots that expose source workbook content
